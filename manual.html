1. Quick Start

  1. Open Fravora and complete onboarding (or replay it later from Settings > Support).
  2. Go to Settings > Sources and add at least one photo source (SMB/NAS, local folder, Google Photos, or OneDrive).
  3. Go to Settings > Albums, create an album, and attach source folders/collections.
  4. In Settings > Slideshow, set Photo Interval, Display Mode, and optional Running Hours.
  5. Save settings and return to the frame view. The slideshow starts once images are available.
Tip: If nothing appears, check the in-app guidance cards first. They explain missing albums, missing sources, or sources with no images.

2. Settings: Sources

Use Sources to define where photos come from.

2.1 Network Shares (SMB/NAS)

  • Add a share using host/share format (no smb:// prefix).
  • Optionally provide username and password.
  • Pick one or more folders from the share. A share with no selected folders is inactive.
  • Use Network Retry Count for unstable networks.

2.2 Local Folders

  • Add local folders with the system picker and grant folder access.
  • Albums can reference multiple local folders (PRO) or one folder (Free).

2.3 Cloud Sources

  • Supported providers: Google Photos and OneDrive.
  • Connect account, then select cloud folders/collections for album use.
  • Google Photos supports picked sets and import actions from album flows.

2.4 Cache and Storage

  • Image Cache Size controls local resized-image cache footprint.
  • Google Photos storage settings reserve free-space headroom to protect device storage.
  • You can clear standard image cache and Google Photos cache from settings actions.

3. Settings: Albums

Albums are the playback containers Fravora scans for slideshow images.

  • Create one or more albums and enable at least one album for playback.
  • Inside each album, attach SMB folders, local folders, and cloud collections.
  • Use Include subfolders when your images are nested (PRO for SMB recursion rules).
  • Use Hidden photos to exclude specific photos and unhide them later.
  • Album counters show visible/hidden/total image counts to verify configuration.
If selected albums contain only hidden images, Fravora will not display photos until some are unhidden.

4. Settings: Slideshow

4.1 Playback controls

  • Photo Interval range: 10 to 600 seconds.
  • Image Order options include Date Added, Taken, Album Order, File Name, Random, and Smart Shuffle (PRO).
  • Display Mode: Fit (no crop) or Fill (screen-filling with possible crop).
  • Transition Effect: Free uses None at runtime; PRO unlocks full effects.

4.2 Running Hours and standby

  • Enable Running Hours for daily start/end windows.
  • Optional custom schedules per day can be configured.
  • Outside running hours, Screen saver shows standby clock or a black screen.
  • Bouncing standby clock is PRO.

4.3 Reliability behavior

  • Fravora retries network image loads based on Network Retry Count.
  • On repeated SMB failures, Fravora can pause and show a connection problem flow with retry/open-sources actions.
  • If some shares are down, slideshow continues with reachable sources when possible.

5. Settings: Appearance and Overlays

5.1 Appearance

  • App Theme: System, Light, or Dark.
  • App Language can follow system or be forced to a chosen language.
  • Fullscreen Mode and Keep Screen Awake improve frame-like behavior.
  • App Color Scheme and advanced sizing controls are PRO-gated.

5.2 Overlays

  • Clock overlay is available on Free and PRO.
  • Weather, Photo Info (metadata), Artist and Song Info, Photo Counter, and Radio Indicator are available with tier gating.
  • Overlay Layout lets you place overlays in corners or hide supported overlays.
  • Avoid Burn-in Effect periodically shifts overlays and adjusts opacity (PRO).

6. Settings: Radio

  • Search for stations and set Current Station.
  • Free tier has search constraints and session-only station persistence.
  • PRO unlocks Favorites, Recently played, and persistent station behavior across restarts.
  • Radio Running Hours and custom daily scheduling are supported (PRO for advanced patterns).
  • Keep Radio Playing in Background and lockscreen/notification controls are PRO.
If radio fails to start, Fravora shows typed failure reasons such as timeout, network issue, access denied, unavailable stream, or unsupported format.

7. Settings: Voice

Voice controls are PRO features and require microphone permission plus an Android speech recognition service.

7.1 Wake phrases

  • Hello Frame
  • Hello Fravora

7.2 Common command intents

  • Play/start/resume radio
  • Stop/pause radio
  • Play/pause/stop slideshow
  • Disable voice control

7.3 Behavior notes

  • Commands are English token-based phrases.
  • If no station is configured and a radio start command is detected, Fravora opens Settings > Radio.
  • You can enable follow-up command windows and wake feedback sound.

9. Troubleshooting

No photos showing

  • Confirm at least one album exists and is selected.
  • Confirm the selected album has active sources attached.
  • If photos are nested, enable Include subfolders where needed.

Only hidden photos found

  • Go to Settings > Albums > Hidden photos and unhide needed images.

Network share issues

  • Verify host/share path and credentials.
  • Increase Network Retry Count for unstable links.
  • Use Open Sources from connection warning dialogs for quick correction.

Google Photos import/storage blocked

  • Free storage space on the device.
  • Lower Google Photos minimum free-space headroom in Sources.
  • Clear Google Photos cache if old picks are no longer needed.

Voice cannot be enabled

  • Install/enable a speech recognition service (commonly Speech Services by Google).
  • Set it as the device voice input service in Android settings.

10. Support

  • Use Settings > Support > Report an Issue to send details by email.
  • Use Feature request for enhancement ideas.
  • Use Check for updates to verify whether a newer app version is available.
